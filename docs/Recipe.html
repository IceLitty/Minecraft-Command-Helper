MCH -> 合成表生成器

操作说明：

· 此合成器中的勾选均为是否带Meta值的选项，即为勾上则会生成Meta值对应输出。

· 右键物品选单可打开物品窗口进行详细的ID/Meta值设定

· 鼠标悬浮物品选单则会看到Meta值

· 结果如同第三方合成表插件类似

· 合成图案会自动进行优化

1.12版本加入此功能，但将json文件放入存档目录不会起作用，只有使用服务端jar插入assets/minecraft/recipes才能起作用。另：此处不会更新合成书里的合成方式

举例：

材料TNT放左侧，白色染色玻璃放右侧，可以合成5块冰块

{
  "type": "crafting_shaped",
  "group": "ICY",
  "pattern": [
    "&^"
  ],
  "key": {
    "&": {
      "item": "minecraft:tnt"
    },
    "^": {
      "item": "minecraft:stained_glass",
      "data": 0
    }
  },
  "result": {
    "item": "minecraft:ice",
    "count": 5
  }
}
