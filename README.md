# Minecraft Command Helper 2
用于方便生成MC高级又复杂指令的程序。

## Features
从生成单个的物品，到药水，乃至烟花火箭

继续涉及到召唤的生物等实体，旗帜生成和Testfor

以及重置过后的JSON文本编辑

再到各种现成就有，好玩有趣的其他指令

应有尽有

集合多种功能，然而这并不是洗衣机

没错，它就是……！

就是啥来着……
